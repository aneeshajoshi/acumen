## Installation and Setup Instructions
#### Clone down this repository. You will need 'node' and 'npm' installed globally on your machine.
## Instllation:
'npm install'
## To create react app:
'npx create-react-app'
## To run Test Suite:
'npm test'
## To Start Server:
'npm start'
## To visit App:
'htp://localhost:3000/'
